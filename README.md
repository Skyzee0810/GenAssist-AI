# GenAssist AI

GenAssist AI is a production-ready AI-powered knowledge assistant built using modern Generative AI technologies.

## Features

- Document Upload
- Retrieval-Augmented Generation (RAG)
- Semantic Search
- Local Large Language Model
- AI Chat
- Source Citations

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- Redis
- LangChain
- LangGraph
- FAISS
- HuggingFace
- Ollama
- React
- Docker
